A very attractive notification popup plugin which gives you a functionality to show your Notification when any user visit your WordPress site.    

== Description ==

A Very effective and attractive plugin which display your Notification to the user in popup.  
* Can set a Background color and text color of Notification popup.  
* Can Place any type of shortcode like contact form in the Notification popup.  
* Can set a time duration of popup generation.
* Can Enable/Disable Notification popup for the pages.

Click Here for Demo http://wptreasure.com/notification-popup/

== Installation ==

* Upload the plugin folder to the /wp-content/plugins/ folder of your WordPress installation.  
* Activate the plugin.   
* You will see an option at the left side WordPress options option panel named "Notification Popup".  
* Click on it to set the options.
Enjoy it......
